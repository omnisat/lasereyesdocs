# Getting Started

## Generate New Project

For new projects, it is recommended to set up your app using the `create-lasereyes` command line interface (CLI).

:::code-group

```bash [npx]
npx create-lasereyes
```

```bash [yarn]
yarn create lasereyes
```

```bash [npm]
npm init lasereyes
```

```bash [pnpm]
pnpm dlx create-lasereyes
```

```bash [bun]
bunx create-lasereyes
```

:::

Once the command runs, you'll see some prompts to complete.

## Adding To An Existing Project

1. Install the required dependencies.

:::code-group

```bash [yarn]
yarn add @omnisat/lasereyes
```

```bash [npm]
npm install @omnisat/lasereyes
```

```bash [pnpm]
pnpm i @omnisat/lasereyes
```

```bash [bun]
bun i @omnisat/lasereyes
```

:::

2. Wrap your app with the `LaserEyesProvider`:

:::code

```jsx [layout.tsx]
import { LaserEyesProvider, MAINNET } from "@omnisat/lasereyes";

function App() {
  return (
    <LaserEyesProvider config={{ network: MAINNET }}>
      {/* Your app components */}
    </LaserEyesProvider>
  );
}
```

:::

2. Use the `useLaserEyes` hook in your components:

```jsx
import { useLaserEyes, UNISAT } from "@omnisat/lasereyes";

export function ConnectWallet() {
  const { connect, connected, address } = useLaserEyes();

  return (
    <div>
      {!connected ? (
        <button onClick={() => connect(UNISAT)}>Connect Wallet</button>
      ) : (
        <div>
          <p>Connected Address: {address}</p>
        </div>
      )}
    </div>
  );
}
```
