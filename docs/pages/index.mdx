# Features
Laser Eyes supports all these features out-of-the-box:
    - Support for multiple Bitcoin wallets: [OYL](https://oyl.io), [Unisat](https://unisat.io), [Xverse](https://xverse.com), and [Leather](https://leather.io)
    - Easy-to-use React context and hooks
    - Network switching between mainnet and testnet
    - Comprehensive wallet operations including balance checking, transaction signing, and message signing
    - TypeScript support
