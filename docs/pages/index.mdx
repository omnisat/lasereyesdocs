# Overview

```tsx
// [!include ~/snippets/app.tsx]
```

Then, use the `useLaserEyes` hook to access the context in any child component.

```tsx
// [!include ~/snippets/ConnectWallet.tsx]
```

Check out [Getting Started](docs/getting-started) for more details.

# Features

Laser Eyes supports all these features out-of-the-box: - Support for multiple Bitcoin wallets: [Leather](https://leather.io), [Unisat](https://unisat.io), and [XVerse](https://www.xverse.app/) - Easy-to-use React context - Network switching between mainnet and testnet - Comprehensive wallet operations including balance checking, transaction signing, and message signing - TypeScript support
